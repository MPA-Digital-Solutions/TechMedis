# Formato de Descripciones de Productos

Este documento explica cómo formatear las descripciones de productos en TechMedis para que se muestren con estilos personalizados.

## Soporta los siguientes formatos:

### 1. Viñetas (Bullet Points)
Usa guion + espacio al inicio de la línea:

```
- Ventaja 1
- Ventaja 2
- Ventaja 3
```

Se mostrará como:
- Ventaja 1
- Ventaja 2
- Ventaja 3

### 2. Listas Numeradas
Usa número + punto + espacio:

```
1. Paso 1
2. Paso 2
3. Paso 3
```

Se mostrará como:
1. Paso 1
2. Paso 2
3. Paso 3

### 3. Párrafos Normales
Solo texto sin prefijo especial:

```
Este es un párrafo normal.
Se puede extender a múltiples líneas.
```

### 4. Separar Secciones
Deja una línea en blanco para separar párrafos:

```
Primera sección con contenido.

Segunda sección con contenido.

Tercera sección.
```

## Ejemplo Completo

```
Descripción general del equipamiento.

Características principales:
- Pantalla de alta resolución 4K
- Procesador de última generación
- Sistema de enfriamiento avanzado

Especificaciones técnicas:
1. Dimensiones: 50x30x20 cm
2. Peso: 15 kg
3. Consumo: 500W máximo

Incluye garantía de 2 años y soporte técnico.
```

## Nota sobre Espacios y Tabulaciones

- ✅ Los saltos de línea se respetan
- ✅ Los espacios en blanco entre párrafos se respetan
- ⚠️ Los espacios múltiples dentro de una línea se normalizan a un único espacio
- ⚠️ Las tabulaciones se convierten a espacios individuales

## Dónde se Usa

- **Página de Detalle del Producto**: Se muestra la descripción completa con todos los formatos
- **Tarjetas de Producto**: Se muestra un resumen truncado (máx 5 líneas)
- **Catálogos**: Se muestran las tarjetas resumidas

## Styling Automático

Todos los formatos se aplican automáticamente con estilos consistentes:
- **Viñetas**: Punto azul (color primario) con margen izquierdo
- **Números**: Números en color primario
- **Párrafos**: Texto estándar con espaciado profesional
